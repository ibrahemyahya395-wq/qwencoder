# 🤖 مساعد المحتوى الذكي

تطبيق ويب عربي بسيط وحديث لإنشاء المحتوى باستخدام الذكاء الاصطناعي.

## ✨ المميزات

- واجهة عربية حديثة وسهلة الاستخدام (RTL)
- إنشاء محتوى ذكي باستخدام Puter AI
- لا يحتاج إلى API Key
- زر لنسخ النص إلى الحافظة
- زر لتحميل الناتج كملف TXT
- تصميم متجاوب يعمل على جميع الأجهزة
- مؤشر تحميل أثناء معالجة الطلب

## 🚀 طريقة التشغيل

### الطريقة 1: فتح الملف مباشرة
1. قم بتنزيل المشروع أو استنساخه
2. افتح ملف `index.html` في أي متصفح حديث (Chrome, Firefox, Edge, Safari)
3. ابدأ في استخدام التطبيق!

### الطريقة 2: استخدام خادم محلي (موصى به)
```bash
# باستخدام Python 3
python -m http.server 8000

# ثم افتح المتصفح على
http://localhost:8000
```

أو باستخدام Node.js:
```bash
npx serve
```

## 📱 كيفية الاستخدام

1. اكتب الموضوع الذي تريد إنشاء محتوى حوله في مربع النص
2. اضغط على زر "✨ إنشاء المحتوى"
3. انتظر حتى يتم إنشاء المحتوى
4. يمكنك نسخ النص أو تحميله كملف TXT

### أمثلة على المواضيع:
- "اكتب لي خطة لتعلم البرمجة في 3 أشهر"
- "لخص لي فوائد القراءة اليومية"
- "أعطني أفكارًا لمشروع ريادي في مجال التكنولوجيا"
- "اكتب مقدمة عن الذكاء الاصطناعي"

## 🛠️ التقنيات المستخدمة

- HTML5
- CSS3 (مع تصميم متجاوب)
- JavaScript (Vanilla)
- [Puter.js](https://js.puter.com/v2/) - للذكاء الاصطناعي بدون API Key
